# Assets

Place your icon/splash images here:

- `icon.png` — 1024×1024 app icon
- `splash.png` — splash screen (1284×2778 or similar high-res)
- `adaptive-icon.png` — Android adaptive icon foreground (1024×1024, transparent bg recommended)
- `favicon.png` — web favicon

If you don't have these yet, Expo will fall back to its defaults — the app will still run, just with a generic icon. You can generate placeholders here: https://easyappicon.com or create them in Figma.
